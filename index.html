验证车牌号是否合法

中文可以使用下面的正则表达式: 


    [\u4e00-\u9fa5]

Object-C示例代码:

- (BOOL) validateCarNo:(NSString *) carNo
{
    BOOL isValid = NO;
    if ([carNo length] != 7) {
        NSLog(@"车牌必须是7位");
    } else {
        NSString *carRegex = @"^[\u4e00-\u9fa5]{1}[A-Za-z]{1}[A-Za-z_0-9]{5}$";
        NSRange range = [carNo rangeOfString:carRegex options:NSRegularExpressionSearch];
        if (range.location != NSNotFound) {
            isValid = YES;
        }
    }
    return isValid;
}

将本地文件添加到github中

第1步: 创建SSH Key

你和Github之间的通信是通过SSH加密的，所以要把公钥告诉Github。 看看家目录有没有.ssh文件夹，该文件夹下有没有id_rsa和id_rsa.pub两个文件。如果有这两个文件，就可以跳到下一步了。

ssh-keygen -t rsa -C "youremail@example.com"

id_rsa是私钥，id_rsa.pub是公钥。

第2步: 上传公钥

在Godaddy上给github pages配置自定义域名

Github pages默认的域名是username.github.io，十分丑陋。

在repository根目录中添加CNAME文件，并添加你的域名。CNAME文件只能有一行:

jingchao.me

打开Godaddy网站，进入“Manage My Domains”:

在Godaddy中配置DNS ZoneFile必须保证Nameserver是Godaddy自己的。在要修改的域名旁边点击向下的三角，然后点击“Set Nameservers”:

保证Nameserver是Standard，即GoDaddy自己的Nameserver:

进入“Manage DNS”:

修改DNS Zone File:

  1. 在“A(Host)”区块下，将IP指向: 192.30.252.153
  2. 在“CName (Alias)”区块下， 将www指向: username.github.io

保存，等待你的DNS配置更新成功。

百度LBS开放平台坐标转换及逆地理编码服务

坐标转换

我获得iphone的定位结果后，在百度地图上进行标注，发现位置有上百米的偏差。是因为百度地图SDK和手机GPS定位使用了不同的坐标系统。

目前常用的坐标系统有三种：

  1. GCJ-02，国测局经纬度坐标。由中国国家测绘局制订的地理信息系统的坐标系统。引入随机的偏差，对经纬度数据进行加密。google地图、soso地图都使用这种标准。
  2. WGS84，为GPS全球定位系统使用而建立的坐标系统。在iOS中，使用CLLocationManager接口定位出来的坐标属于该标准。
  3. bd09ll，百度经纬度坐标。百度地图SDK采用的是百度自有的地理坐标系
