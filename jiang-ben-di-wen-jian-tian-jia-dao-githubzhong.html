将本地文件添加到github中

Date 二 26 八月 2014 Tags github

已经在github页面中新建repository: yourproject

cd ~/localproject
git init
git add .
git commit -m "本地文件添加到github中"
git remote yourproject https://github.com/username/yourproject.git
git push yourproject master
