将本地文件添加到github中

Date 二 26 八月 2014 Tags git / github

第1步: 创建SSH Key

你和Github之间的通信是通过SSH加密的，所以要把公钥告诉Github。 看看家目录有没有.ssh文件夹，该文件夹下有没有id_rsa和id_rsa.pub两个文件。如果有这两个文件，就可以跳到下一步了。

ssh-keygen -t rsa -C "youremail@example.com"

id_rsa是私钥，id_rsa.pub是公钥。

第2步: 上传公钥

登录Github -> Settings -> SSH Keys -> Add SSH Key，输入Title及id_rsa.pub文件内容，点击Add key，如下图:

第3步: 第一次push文件

在github页面中新建repository: yourproject

cd ~/localproject
git init
git add .
git commit -m "本地文件添加到github中"
git remote add <alias-name> https://github.com/username/yourproject.git
git push -u <alias-name> master

在第一次push之前，远程库还是空的。加上-u参数不但可以把本地master分支的内容同步到远程master分支，还可以把本地master分支和远程master分支关联起来，方便以后push。之后的push就可以用下面的命令:

git push <alias-name> master

alias-name是你为远程库起的一个别名，git默认是“origin”。如果你将一个远程库clone到本地，那么远程库默认会使用origin作为别名。
