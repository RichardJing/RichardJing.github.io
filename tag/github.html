将本地文件添加到github中

第1步: 创建SSH Key

你和Github之间的通信是通过SSH加密的，所以要把公钥告诉Github。 看看家目录有没有.ssh文件夹，该文件夹下有没有id_rsa和id_rsa.pub两个文件。如果有这两个文件，就可以跳到下一步了。

ssh-keygen -t rsa -C "youremail@example.com"

id_rsa是私钥，id_rsa.pub是公钥。

第2步: 上传公钥

登录Github -> Settings -> SSH Keys -> Add SSH Key，输入Title及rsa文件内容，点击Add key，如下图:

在Godaddy上给github pages配置自定义域名

Github pages默认的域名是username.github.io，十分丑陋。

在repository根目录中添加CNAME文件，并添加你的域名。CNAME文件只能有一行:

jingchao.me

打开Godaddy网站，进入“Manage My Domains”:

在Godaddy中配置DNS ZoneFile必须保证Nameserver是Godaddy自己的。在要修改的域名旁边点击向下的三角，然后点击“Set Nameservers”:

保证Nameserver是Standard，即GoDaddy自己的Nameserver:

进入“Manage DNS”:

修改DNS Zone File:

  1. 在“A(Host)”区块下，将IP指向: 192.30.252.153
  2. 在“CName (Alias)”区块下， 将www指向: username.github.io

保存，等待你的DNS配置更新成功。
